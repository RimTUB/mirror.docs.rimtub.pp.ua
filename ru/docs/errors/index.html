Частые ошибки

Общее​

  • ImportError: cannot import name 'Self' from 'typing'​

    У тебя установлена старая версия питона. Скачай 3.11

  • sqlite3.OperationalError: unable to open database file​

    Создай папку sessions в папке ЮБ

  • AttributeError: 'Config' object has no attribute 'BOT_TOKEN'​

    Ты не указал(а) в config.yaml токен бота.

  • AttributeError: 'Config' object has no attribute 'PHONES'​

    Ты не указал(а) номера телефонов в config.yaml

  • FileNotFoundError: [Errno 2] No such file or directory: '...\storage\nYlI2wkF6G.pkl'​

    Создай папку storage в папке ЮБ

  • pyrogram.errors.exceptions.bad_request_400.BotInlineDisabled: Telegram says: [400 BOT_INLINE_DISABLED] - The inline feature of the bot is disabled (caused by "messages.GetInlineBotResults")​

    Ты не включил(а) Inline mode у бота в @BotFather

  • error: subprocess-exited-with-error​

    Вариантов несколько

    • Если у тебя Termux, то обнови его, скорее всего он старой версии.
    • Если ты на своем ПК/Ноуте, то скорее всего у тебя не та версия Python. Установи 3.11

  • sqlite3.OperationalError: database is locked​

    Убедись что юб больше нигде не запущен. Если не помогло, попробуй это:

    • Windows
    • Linux
    1. Скачай и установи handle:
      1. Скачай Handle.zip
      2. Распакуй все файлы в C:\Windows\System32
    2. cd sessions
    3. handle RimTUB.session
    4. taskkill /f /pid 8388, где 8388 это pid которые нашлись выше
    1. cd sessions
    2. sudo apt install lsof -y
    3. lsof RimTUB.session
    4. kill -9 2276290, где 2276290 это pid что нашлись выше

MavisHost​

  • /usr/local/bin/python: can't open file '/home/container/main.py' : [Errno 2] No such file or directory​

    Ты не переместил(а) файлы ЮБ в корень. Перейди в папку RimTUB, выбери все файлы, нажми кнопку Move и перемести по пути .. (две точки)

  • /usr/local/bin/python: can't open file '/home/container/App.py' : [Errno 2] No such file or directory​

    Ты не установил(а) PY APP FILE во вкладке Startup. Установи его на main.py
